# Forge Build

## About

A 4 week workshop for teaching basics of web3, especially Blockchain and Smart Contracts

## Week 1

- Task 1 : Creating Basic Skeleton for a Personal Website using HTML
- Task 2 : Styling the website using CSS
- Task 3 : Basic JS task -
  - Adding an interactive counter
  - Form that shows submission
  - Toggling Themes
- Task 4 : Creating a SimpleStorage contract
